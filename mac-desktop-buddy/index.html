MacDesktopBuddy

把一只可拖动的小企鹅放在桌面边角，帮你记待办、按时提醒、适时休息，还能用语音快速添加任务。

一个轻巧，但很会照顾节奏的桌面工具。

MacDesktopBuddy 不抢你的注意力，只在需要时出现：记录任务、弹出气泡、播放音效，然后把空间还给你。

✓

待办清单

添加、完成、恢复和删除任务，长文本会完整显示，完成后自动移到已完成列表。

⏰

任务提醒

支持 15 分钟、30 分钟、1 小时和自定义时间，到点在桌面图标旁弹出气泡并播放音效。

☕

休息提醒

按你设置的间隔提醒休息，提示语温柔、俏皮，也可以安静地从设置里开关。

🎙

语音输入

基于 macOS 语音识别，中文为主，也支持中英夹杂的任务口述。

◎

隐身模式

右键或长按小Q即可隐身，需要恢复时从菜单栏入口唤回。

⌂

本地保存

待办数据保存在本机，不依赖后端服务，也没有上传任务内容的代码路径。

安装很简单。

下载 DMG

从 GitHub Release 获取最新版 MacDesktopBuddy.dmg。

拖到 Applications

打开 DMG，把 MacDesktopBuddy.app 拖到应用程序文件夹。

授予权限

首次使用语音输入时，按系统提示允许麦克风和语音识别权限。

它很近，也很安静。

本地优先

待办保存到本机 Application Support 目录，设置保存在 UserDefaults。

只在需要时请求权限

麦克风和语音识别仅用于语音输入；通知权限用于系统提醒。
