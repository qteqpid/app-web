看清浏览器时间都花在哪。

Browser Time Tracker 统计你在各个网站和网页上的停留时间， 用一个本地 Dashboard 展示 Chrome、Safari、Edge 和 Firefox 的使用分布。

它能做什么

比系统屏幕使用时间更细的网页视角。

macOS 屏幕使用时间只能告诉你 Chrome 打开了多久。Browser Time Tracker 会继续细分到： 哪些网站、哪些页面、哪一天、哪一个小时。所有数据都留在你的 Mac 本地。

01

按网站和页面统计

按域名和页面标题记录使用时间，在本地 Dashboard 里查看 Top Websites 和 Top Pages。

02

日级和小时级图表

查看最近七天，点击某一天继续下钻到小时分布，并用颜色区分不同网站。

03

为菜单栏而生

用一个小沙漏图标安静运行；睡眠、锁屏时自动暂停，并只保留最近一周数据。

开始使用

一分钟装好。

  1. 1
    下载 DMG

    从最新 GitHub Release 获取 BrowserTimeTracker.dmg。

  2. 2
    拖到 Applications

    打开 DMG，把 Browser Time Tracker.app 拖进 Applications 文件夹。

  3. 3
    授权浏览器权限

    允许 Chrome、Safari 或 Edge 的自动化权限；如需统计 Firefox，再开启辅助功能权限。

  4. 4
    打开 Dashboard

    点击菜单栏里的沙漏图标，选择 Open Dashboard。

默认本地优先

浏览数据只留在你的 Mac 上。

不需要云账号

不需要注册、同步服务或托管后端。

本地 SQLite 存储

使用记录存放在 Mac 的 Application Support 目录下。

只保留七天

超过一周的旧 session 会自动清理。

Admin 锁

Pause、Clear Data 和 Quit 可以要求管理员鉴权。
